# 팀원 소개 페이지

Git 커밋 협업을 통한 팀원 소개 페이지 프로젝트입니다.
충돌 테스트 하기

![main image](image/main_old.png)

```bash
// 팀장은 각 절차 전에 필요한 사항들을 사전에 준비하고 예회상황을 체크한다
// 강의 진도는 각자의 속도에 맞춰서 하되 과제 진도는 팀원 전체가 맞춰서 실습 상황을 경험 할 수 있도록 진행

0. 우선 강의를 들으면서 기본 지식 습득 (중간 중간 서로 공유 17시까지)
1. 17시 부터 깃허브 회원 가입 진행 밎 프로젝트 초대 (강의 들으면서 깃 설치를 안했을 시 깃 설치 진행)
2. 7시에 저녁 먹고와서 다 같이 클론을 내려 받는다 (인증관련 이슈 예상, 같이 해결 해볼 생각)
3. 클론 완료 후 각자 브랜치까지 만들고 체크아웃 진행 (안전한 작업 환경을 먼저 만들어 둔다) 
4. members 폴더에 각자 파일 하나씩 만들고 example.json 내용을 복사하여 맴버카드 내용 작성
 - 작업 예시등 작업 요령 관련하여 회의로 공유
5. 맴버 카드들에 대한 리퀘스트가 들어오면 팀장은 확인 후 머지 main에 반영한다
6. 각 팀원의 작업이 반영된 프로젝트를 팀장은  members.json 수정 작업 진행 및 기타 프로젝트 정리를 진행하여 다시 메인에 반영
// 여기까지 1월 6일에 끝내느 것을 목표로 진행
7. 1월 7일 오전에 각자 main 브랜치에 같은 파일을 수정하는 push를 날리는 실습을 진행 브랜치를 사용하지 않는 충돌 상황을 경험해 본다
8. 브랜치를 사용하여 충돌 상황을 연출한다 팀장은 화면을 공유하여 팀원들과 이야기 하면서 충돌을 해결하는 상황을 같이 경험하도록 한다
9. 다같이 리베이스를 해본다
10. commit --amend 각자 해본다
```

### 템플릿별 역할 안내
- **member1.json** (김개발): 프론트엔드 개발자 - React, TypeScript 중심
- **member2.json** (이백엔드): 백엔드 개발자 - Node.js, Python 중심
- **member3.json** (박디자인): UI/UX 디자이너 - Figma, Adobe XD 중심
- **member4.json** (최풀스택): 풀스택 개발자 - React, Node.js 모두
- **member5.json** (정데이터): 데이터 분석가 - Python, SQL, 머신러닝 중심
